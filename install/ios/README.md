# iOS

## 介绍

苹果手机的代理软件客户端推荐QuantumultX以及Shadowrocket这两个客户端。其中**个人更推荐QuantumultX**，可提供更加详细的分流，与Clash使用方式更加贴近。这两款软件都无法在中国区App Store下载，需切换到美国或香港App Store进行下载且都需要付费（只有iOS客户端需要付费）
